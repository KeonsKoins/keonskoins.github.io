# KeonsKoins GitHub Pages Website

Free landing page for KeonsKoins with dropdown menus and links to the eBay store.

## How to publish on GitHub Pages

1. Create a free GitHub account or sign in.
2. Click **New Repository**.
3. Name it `keonskoins` or `keonskoins-site`.
4. Upload these files:
   - `index.html`
   - `style.css`
   - `script.js`
5. Go to **Settings > Pages**.
6. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
7. Save.
8. GitHub will give you a free website link.

Your eBay store link is already connected: https://www.ebay.com/usr/keonskoins
