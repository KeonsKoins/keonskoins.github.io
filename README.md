# KeonsKoins Version 2

Static GitHub Pages preview for KeonsKoins.

## What's included
- Polished homepage hero
- Live-style Today's Bullion Prices ticker placeholder
- eBay store calls-to-action
- Collector categories
- Roll Vault section
- Collector Club signup placeholder
- We Buy section
- Mobile responsive navigation

## Notes
The bullion ticker currently uses preview values in `script.js`. To make it truly live later, replace the preview object with a metals API or embedded bullion widget.
